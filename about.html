Web Designer

Web designer refers to the the design of websites that are displayed on the internet. It is usually refers to the user experience aspects of website development rather than the software development.
A web designer works on the apperance,layout and in some case content website.

What Web designers do?

Web design identifies the goals of a website or webpage and promotes accessibility for all potential users. This process involves organizing content and images across a series of pages , integrating applications and other interactive elements. The professionals who perform this process are called web designers and their job includes the following duties.

  1. selecting easy to read fonts
  2. choosing attractive colors schemes that also enables easy to read fonts
  3. Implemenyting brsnds identity in to the colours , fonts and layouts
  4. creating map of the website structure to ensure intutive navigation
  5. Placing images,logos,text,video application and other elements
  6. Using coding languages sucha as HTML and CSS to create layouts and to style pages
  7. Making optimized versions of websites and pages both for desktop and mobile viewing
